# Existing Articles Workflow

Use this area for blog content that already exists before the new workflow in this repository.

## Process

- Import the original article into `raw/`
- Optionally normalize or clean it into `cleaned/`
- Perform SEO or content review into `audits/`

## Purpose

Existing articles in this area can be used as reference material for:

- future article writing
- internal linking planning
- topic gap analysis
