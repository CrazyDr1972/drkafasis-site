# drkafasis-site

Umbrella workspace repository for the full Dr Kafasis website.

## Workspace areas

- `theme/` contains the existing WordPress theme work and audit snapshot.
- `blog/` contains content planning, drafts, published material, templates, and SEO workflow files.
- `site/` contains broader site documentation, assets, redirects, schema, and structure notes.
- `automation/` contains scripts and future WordPress API publishing tools.

## Current theme workspace

The existing theme repository has been copied into:

- `theme/drkafasis-betheme-audit/`

Its contents are now regular files inside this umbrella repository rather than a nested Git repository.

## Scope

This repository is intended to become the main local workspace for:

- WordPress theme work
- blog content and SEO operations
- site-wide documentation
- future safe automation and publishing support
