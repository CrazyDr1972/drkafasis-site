# AGENTS.md

## Workspace rules

- Never mix blog content files into the theme code folders.
- Treat `theme/`, `blog/`, `site/`, and `automation/` as separate domains.
- Prefer minimal, targeted changes.
- For content tasks, work under `blog/`.
- For WordPress code tasks, work under `theme/drkafasis-betheme-audit/`.
- For future publish/integration tooling, work under `automation/`.
- Ask for approval before any destructive move or deletion outside the new umbrella repo.
- Do not rewrite or reorganize the copied theme code unless explicitly asked.
